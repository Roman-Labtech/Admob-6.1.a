Расчет площади треугольника

Данный калькулятор поможет легко рассчитать площадь треугольника по извесным величинам.

По основанию и высоте Формула нахождения площади треугольника по основанию и высоте. По трем сторонам Формула Герона для нахождения площади треугольника по трем известным сторонам. По 2-м сторонам и углу Формула нахождения площади треугольника по двум сторонам и углу между ними.
